## Tomayon  (WIP)

A Harp boilerplate using my favourite color: Tomato and my favourite CSS tool: Tachyons.

Color Scheme

Primary: RGBA(255, 99, 71, 1) (tomato)     
Secondary: rgb(98, 71, 255)  
Background: rgb(255, 255, 255)  
Foreground: rgb(51, 51, 51)
